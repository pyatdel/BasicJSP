# 문제 설정 방법
 - egovFramework를 실행하세요.
 - JSPBook 프로젝트 안에서 webapp 폴더 안에 resources 폴더를 넣어주세요.
 - JSPBook 프로젝트 안에서 webapp 폴더 안에 pageModule 폴더를 넣어주세요.
 - JSPBook 프로젝트 안에서 webapp 폴더 안에 script 폴더를 만들고 그 안에 question01.jsp를 넣어주세요.
 - JSPBook 프로젝트 안에서 webapp 폴더 안에 script 폴더를 만들고 그 안에 question02.jsp를 넣어주세요.

1. question01.jsp 
 - 일반 게시판이 있습니다. 
   해당 게시판에서 각 td영역을 클릭했을때, 내가 클릭한 영역의 텍스트가 출력란에 나타나게 해주세요.
 - 출력란에 나타낼때에는 내가 클릭한 영역의 텍스트와 [수정]버튼, [삭제]버튼을 함께 출력해주세요.
 - 수정 버튼을 클릭 시, 수정 폼이 출력되어 수정할 수 있게 해주세요.
 - 수정이 완료되었다면 수정이 완료된 텍스트로 결과를 반영해주세요.
 - 삭제 버튼을 클릭 시, 해당 영역의 텍스트가 삭제되게 해주세요.
 - append 여부를 체크 후, td 영역을 클릭하게되면 출력란에 결과가 계속 아래로 붙으면서 출력되게 해주세요.

2. question02.jsp
 - 401호 자리표가 있습니다. 
 - 첫번재 학생을 클릭하고 두번재 학생을 클릭하면 자리가 서로 변경되게 해주세요.
 - 출력란에 첫번재 학생을 클릭하고 두번재 학생을 클릭했다는 내역을 남겨주세요.
 - 출력 버튼을 클릭하면 바뀐 자리가 그대로 아래에 출력되게 해주세요.
